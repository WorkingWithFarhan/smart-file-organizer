# Smart File Organizer

A simple Python automation tool that organizes messy folders automatically.

## Version
V0.1

## Features

- Detects file extensions
- Creates category folders automatically
- Moves files into correct folders

Example:

Before:

Downloads/
image.png  
resume.pdf  
movie.mp4  

After:

Downloads/
Images/  
Documents/  
Videos/

## How to Run

1. Install Python
2. Clone repository
3. Run:

python organizer.py

4. Enter folder path when asked.
